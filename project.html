Draw App

The draw app is a full stack application which uses mern stack. So basically we can draw anything we want, colour it ,paint it.The drawing interface is made using HTML5 canvas. Then we can share it onto the website ,where everyone can view your drawing and like it. All the drawings are strored in a mongoDB database. This has been deployed to heroku but later removed since heroku changed its poilicy.

Comet Galleria

Comet Galleria is a place where you can share your photo clicks with the internet. It uses mern stack. The app retrives all the images in a mondoDB server and displays in onto the webpage.These photos can be liked, disliked and also can be commented on. This was one of my first full stack applications.

Artistry

Since I was quite enthusiatic about art, I wanted to create a platform for me and my fellow artists to show their skills in this application. Here on Artistry, you can post your drawings and meet other like minded artists. Similar to Comet Galleria, all the posts can be criticized. There was an intuition to host a platform where you can even sell your artworks but it hasnt been implemented yet onto artistry.
